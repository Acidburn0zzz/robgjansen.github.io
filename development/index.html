Toward Anonymity in Delay Tolerant Networks: Threshold Pivot Scheme

This project develops ideas for anonymity in delay/disruption tolerant networks. The simulation code for the TPS scheme was customized from The ONE Simulator. Please contact me with questions.

Recruiting New Tor Relays with BRAIDS

This project develops ideas about creating incentives for Tor clients to run Tor relays. We created a custom Tor simulator to analyze the effects of new scheduling mechanisms.
