Development Projects

I have worked on several development projects over the course of my research. Most significantly is my ongoing work on the Shadow network simulator, which I design, develop, and maintain. Shadow is an open source project hosted on GitHub, and can be used to run real applications over simulated networks. Although originally developed to run the Tor anonymity software, Shadowâ€™s plug-in architecture makes it ideal for distributed applications in general.

I have recently developed libkqtime, an open source library that tracks the queuing time of bytes traveling through the Linux kernel. The libkqtime library uses libpcap to intercept packets entering and leaving a machine, and function interposition to intercept packets being read and written by the application. It was orignally written to track congestion and performance problems in Tor, but I hope it will be more generally useful.

More generally, I often hack on Tor as part of my research. When appropriate, those hacks are made available in my GitHub clone of Tor.

Finally, in the more distant past I hacked anonymity mechanisms into the ONE simulator for delay tolerant networks, and wrote my own Tor+BRAIDS simulator to analyze the effects of incentives on Tor network performance.
